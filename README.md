# 说明书

## 1. 运行环境及应用软件

开发技术：JSP 

服务器：Apache Tomcat 9.0

数据库：Mysql 8.0

开发工具：eclipse

界面制作：CSS



## 2. 主要功能

#### 2.1 前台博客系统

- 用户登录注册
- 用户个人信息修改、更换头像
- 发布文章、编辑文章、设置文章权限
- 评论功能

#### 2.2 后台管理系统

- 文章发表、编辑
- 用户管理
